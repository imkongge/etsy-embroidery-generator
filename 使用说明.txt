========================================
Etsy 刺绣预览生成器 - 使用说明
========================================

【首次使用 - 必读】

1. 安装 Python (如果电脑上没有)
   下载地址: https://www.python.org/downloads/
   安装时勾选: Add Python to PATH

2. 安装依赖 (首次运行时会自动安装)
   手动安装命令:
   pip install streamlit pillow openai

3. 配置 Photoshop (自动完成)
   运行"一键启动.bat"会自动禁用PS的脚本警告

========================================

【启动程序】

双击运行: 一键启动.bat

打开浏览器访问: http://localhost:8502

========================================

【使用方法】

1. 输入刺绣名字 (如 Emma)
2. 选择毛衣颜色
3. 选择单色名字 或 彩色名字
4. 选择毛线颜色
5. (可选) 勾选"是否有配件"来自动打开PS合成
6. 点击"开始生成"

========================================

【输出位置】

生成图片保存在:
final_psd/日期/名字_颜色.jpg

示例: final_psd/2026-03-26/Emma_粉色.jpg

========================================

【常见问题】

Q: 提示"未检测到Python"
A: 请先安装Python 3.8+

Q: PS弹出"是否运行脚本"警告
A: 运行"一键启动.bat"会自动处理，重启PS后生效

Q: PS合成功能不正常
A: 目前PS功能仍在调试中，可暂时不勾选

========================================

【分享给同事】

直接把整个文件夹复制给同事即可。

同事需要:
1. 安装Python
2. 双击"一键启动.bat"
3. 浏览器打开 http://localhost:8502
